# Business Canon

These docs explain why Ordo exists, how the product should be judged, and why
the project is shaped as an open appliance for observable AI work instead of a
conventional SaaS product.

The short version:

Ordo is a governed workforce substrate for turning human judgment into durable
business motion.

Chat is where the owner directs work, reviews results, gives feedback, and
approves outcomes. The system behind chat remembers context, routes work,
keeps evidence, and helps the owner decide what should happen next.

These docs are product guidance. They are not all shipped features.
