# Business Canon

These docs explain why Ordo exists and how the product should be judged.

The short version:

Ordo is a governed workshop for turning human judgment into durable business
motion.

Chat is where the owner works. The system behind chat remembers context, routes
work, keeps evidence, and helps the owner decide what should happen next.

These docs are product guidance. They are not all shipped features.
