## Agent Process Spine

Future AI-assisted work should use the Definition Of Done, issue templates, and
Agent Execution Protocol as the process source of truth. The reusable prompts in
`prompts/` are intentionally shorter than older stacked prompts because they
reference these docs instead of restating every rule.
